Nature Quizzer
--------------

Simple web-based adaptive learning system for teaching users biological concepts such as plants, animals,...

Installing
==========

Install development dependencies via `node`:

```
$ node update
```

Run `gulp` to build css, templates and javascript files:

```
$ gulp
```
